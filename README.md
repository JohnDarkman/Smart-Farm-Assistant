CS Guru Project: Smart Farm Assistant

Description: A browser-based AI chatbot that helps gardeners and farmers with personalized advice, seasonal tips, and quick access. This project was developed as part of the Telebort AI-3 Program. This repository contains my own implementation, enhancements, and customizations based on the course.

Features: User onboarding, LocalStorage for preferences and chat history, keyword-based AI responses, quick topic buttons, seasonal advice, responsive design.

Setup and Installation: Clone the repository, open index.html in a browser or use VS Code Live Server / Python HTTP server.

Technical Details: Built with HTML, CSS, JavaScript; uses LocalStorage for data persistence; responsive design.

Contributing: Open issues or submit pull requests for improvements.
